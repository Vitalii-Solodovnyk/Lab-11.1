
Opis projektu
Zrefaktoryzowany system bankowy (System Bankowy 2.0), który zastępuje proste tablice strukturami danych. Program umożliwia zarządzanie kontami klientów, rejestrację historii transakcji oraz zapewnia trwałość danych dzięki operacjom na plikach.
